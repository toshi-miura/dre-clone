このソフトは、お使いのパソコンに保存された画像の中から、類似した画像を検出して処理を行うソフトです。

* 特徴
  - ソフト自体について
    - 検出対象はファイル・フォルダ・ZIPアーカイブ単位で指定できます。複数フォルダの相互比較も可能です。
    - 検出アルゴリズムは、パッケージプラグインとしてあとから追加できます。
    - 検出された類似ファイルは、候補をプレビューし、条件によって選別したあと、削除・移動を行うことができます。

  - デフォルトの検出アルゴリズムパッケージ（Gray64）について
    - 画像の大きさ・色相を無視した類似判定を行います。
    - キャッシュを使用した高速な読み込みができます。特徴量はファイルあたり80byteと小さいので、スペースを取りません。

* 動作環境
  - JVM on Windows XP / Vista / 7
    -- Java製なので、ライブラリを差し替えるだけで Mac や Linux で動作すると思います。需要があれば一報を。

* 使い方
  ** 検出対象の選択画面
    - 左右のファイルツリーから、検出対象のファイルと、比較元のファイルを選択します。
    - 新たに加わったファイルとデータベースフォルダとの間で比較する場合、左では検査したいファイル、右ではデータベースフォルダを指定します。
    - 単一フォルダ内の重複検索などの場合は、「検出対象と同じファイルを比較元にする」にチェックを入れてください。
    - ZIPアーカイブはディレクトリと同じ扱いになります。

  ** 検出アルゴリズムの選択
    - 左側のリストから、使用する検出アルゴリズムを選択します。
    - パッケージごとに、オプションを指定することができます。パッケージのマニュアルに従って設定してください。
    - 「ワーカースレッド数」で、検出処理時に使用するスレッド数を指定できます。
    - 「次へ」を押すと、検出処理が開始します。

  ** 類似判定の実行
    - 処理の進行状況が、プログレスバーとテキストで表示されます。
    - 処理が完了すると、「次へ」ボタンをクリックできるようになります。

  ** 処分するファイルの選択
    - チェックボックスを使用して、削除/移動処理を行うファイルを選択します。
    - 左上は、検出対象ファイルの中で、類似画像の候補があるものがリスト表示されます。
      -- より近い候補があるものが上になるようにソートされます。
      -- 左上のファイルをひとつ選択すると、左下に類似画像候補が表示されます。
      -- ファイルを選択した状態で、スペースバーを押してチェックできます。
      -- シフトキーを押しているあいだ、類似画像候補リストにフォーカスが移ります。
    - 左下は、類似画像候補のリストです。
      -- 一番上に検出対象ファイルが表示され、二番目以降に類似画像候補が表示されます。
      -- ファイル名の右の数字は、検出対象ファイルとの距離です。
      -- ファイルを選択すると、右側にプレビューが表示されます。
      -- ファイルを選択した状態で、スペースバーを押してチェックできます。
      -- シフトキーを離すと、類似ファイルの存在するファイルリストにフォーカスが移ります。
    - 右の「プレビュー」タブをクリックすると、プレビュー画面が表示されます。
      -- 画像は枠に合わせてスケーリングされます。
      -- 「ルーペ」にチェックすると、拡大鏡が有効になります。
      -- 「ルーペ」チェックボックスの右側で、ルーペの拡大率と半径を指定できます。ルーペの拡大率は原寸比です。
      -- ルーペはマウスのドラッグで移動できます。
    - 右の「一括チェック」タブをクリックすると、一括チェック画面が表示されます。
      -- 条件を指定し、「実行する」ボタンを押すことで、一括チェックを実行できます。
      -- 一般的な目的に対する手順は次のとおりです。
        1. 「距離がxx以下のファイルをチェックする」を実行します。
        2. 「（残したいファイルの条件）のチェックを外す」を実行します。 
      -- 「圧縮率」は、画素数 / ファイルサイズです。jpegなどでは、この値が大きいほど強く圧縮され、劣化している可能性があります。

  ** 処分方法の選択
    - チェックしたファイルの処分方法として、削除するか移動するかを選択します。

  ** ファイルの処分処理実行
    - 処理の進行状況が、プログレスバーとテキストで表示されます。
    - 処理が完了すれば、作業は終了です。

* 検出アルゴリズムのプラグインパッケージ
  - packagesの中にjarアーカイブを追加することで、外部のプラグインパッケージを追加できます。
  - 1つまたは2つのクラスファイルとパッケージマニフェストをjarで固めて、プラグインパッケージを作成できます。
  - サンプル用に BinDup (Sample) パッケージを用意しました。ソースコードも同梱しています。
